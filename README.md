## Introduction

**SVG** (Scalable Vector Graphics) is an XML-based format for vector images. Unlike raster images (like PNG or JPEG), SVG images are resolution-independent, meaning they can scale to any size without losing quality. This makes them perfect for responsive web design.

SVG is widely used for things like:

- Icons
- Logos
- Charts
- Illustrations

This tutorial covers the basics of SVG, including how to embed them in HTML, style them with CSS, and make them interactive.

---

## Why Use SVG?

SVGs have several advantages over other image formats:

- **Scalable**: SVGs are resolution-independent, meaning they look great at any size, even on high-resolution screens (e.g., Retina displays).
- **Interactivity**: SVG elements can be styled with CSS and manipulated with JavaScript, making them interactive.
- **Accessibility**: SVGs can be made accessible by adding `aria` labels or using `<title>` and `<desc>` tags.
- **File Size**: SVG files are typically smaller in size compared to raster formats, helping improve page load speed.

---

Here is a simple SVG element:

```html

  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="yellow" />
</svg>

# Event Hub Website

This project is a simple HTML structure for an **Event Hub** website that provides information about both upcoming and past events. The website is structured with a clean layout and includes sections for upcoming events and past events, featuring descriptions, dates, and images for past events.

## Features

- **Navigation**: 
   - Links to sections for *Upcoming Events* and *Past Events*.
   - Easy-to-navigate structure for a better user experience.

- **Upcoming Events**: 
   - Lists future events like the "AI & Machine Learning Conference 2024" and the "Web Development Bootcamp."

- **Past Events**:
   - Displays events such as the "Cybersecurity Summit 2024" and "Blockchain Expo 2024" with relevant images and event details.

## Structure

1. **Header**
   - The website's title is prominently displayed.
   - A navigation menu allows easy access to upcoming and past events.

2. **Main Content**
   - **Upcoming Events**: Two events are listed with their details, including descriptions and dates.
   - **Past Events**: Displays two past events with descriptions, dates, and accompanying images.

## Technologies Used

- **HTML5**: Basic structure for the webpage.
- **CSS** (optional for styling, not included here): To be added for layout and design enhancements.
  
## How to Use

2. Open the `index.html` file in any web browser.
3. The site is fully responsive and will scale based on device size.
